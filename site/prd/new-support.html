建图与导航工具 V1.0.0 项目需新增支持子需求清单

来源：基于“PRD 有，72 版本导览项目目前没有”的差异整理。
用途：给开发评估哪些能力需要在现有项目上新增支持。
不包含：72 版本导览项目已有但本期 PRD 不采纳的能力，例如多点巡航、任务编排、导航策略配置、调试图层、回充动作。

1. 新建地图与建图前置

子需求 对应 PRD 新增支持内容 验收口径 优先级
建图前选择栅格精度 F02-06 新建地图时选择本次建图栅格精度，开始建图后不可修改。 用户开始建图前能选择精度；建图开始后不可再改。 P0
建图前检查 F03 检查控制、底盘 / 急停、扫描数据、建图服务、保存能力。 通过时进入建图工作台；失败时提示失败项和处理动作。 P0

2. 建图保存闭环

子需求 对应 PRD 新增支持内容 验收口径 优先级
保存地图 F05-01 完成建图后保存为正式地图。 保存成功后地图进入地图列表。 P0
保存并编辑 F05-02 保存成功后直接进入地图编辑页。 用户无需回到列表即可继续修图或设点。 P1
保存前校验 F05-03 保存前校验地图有效性、名称、文件写入和阻断异常。 不可保存时停留建图工作台并提示原因。 P0
保存失败重试 F05-05 保存失败时保留未保存地图，允许重试或取消。 失败后不丢失本次建图结果。 P1

3. 地图查看与维护入口

子需求 对应 PRD 新增支持内容 验收口径 优先级
独立地图查看页 F06 点击正常地图进入只读查看页。 查看页展示底图、点位、机器人位置；不允许移动机器人。 P0
查看页维护入口 F06-04 查看页提供编辑、设置点位、下载、返回列表。 用户可从查看结果继续维护或交付。 P1

4. 基础点位资产

子需求 对应 PRD 新增支持内容 验收口径 优先级
点位资产存储 F08 点位作为地图资产保存，不作为导览任务点或路线点。 点位列表与地图绑定，可保存、读取、下载、返回给业务 App。 P0
当前位置设点 F08-04 使用机器人当前位置新增点位，朝向默认取机器人当前朝向。 每点击一次生成一个带坐标和朝向的点位。 P0
地图选点设点 F08-05 点击可通行区域生成点位，长按拖拽设置朝向。 点位必须有坐标和朝向后才可保存。 P0
点位合法性校验 F08-11 校验点位是否在可通行区域、是否有坐标和朝向。 点位在障碍物、没扫到、地图外或未设置朝向时禁止保存。 P0

5. 单点导航验证

子需求 对应 PRD 新增支持内容 验收口径 优先级
选择点位导航 F11-01 / F11-02 用户选择已保存点位，发起导航到点位。 机器人按点位坐标和朝向前往目标。 P1
导航状态和停止 F11-03 执行中展示状态，用户可停止。 停止后机器人停止本次验证并提示状态。 P1
导航异常处理 F11-04 / F11-05 障碍物、手柄接管、定位丢失、急停、连接断开、无法规划路径时停止验证并提示原因。 异常不修改点位数据，用户可处理后重试。 P1

6. 下载与业务 App 调用

子需求 对应 PRD 新增支持内容 验收口径 优先级
下载地图包 F09 下载地图文件、点位列表和必要元数据。 下载包包含点位坐标和朝向；下载失败时可重试。 P0
业务 App 选图返回 F10 自动回充 / 导览 App 调起建图工具选择地图，并接收返回结果。 返回地图 ID、名称、文件状态、最近保存时间、点位数量和点位更新时间。 P0
