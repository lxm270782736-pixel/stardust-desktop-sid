第一部分：需求分析（Why & What）

1. 目标与范围

建图与导航工具用于让当前机器人生成本地地图，并支持用户设置带朝向的点位、导航到点位进行验证，再把地图交给二开下载、自动回充功能选择、导览 App 选择和后续地图维护使用。

本期由于开发资源限制，不做自动建图；不做路线、讲解、回充动作或导航策略配置。

本期导航只做单点验证：用户选择一个点位，让机器人前往该点位并对齐朝向；验证结果不生成路线，不改变点位数据。

2. 用户任务

用户任务 用户要完成什么 完成后得到什么
建图并下载 用户新建地图，手动遥控机器人完成建图，保存后下载地图包。 地图文件、点位列表和必要元数据，可用于开发、调试或导入自有系统。
业务 App 选图 用户从自动回充 App 或导览 App 进入建图 App，选择当前机器人可用地图；没有地图时先新建。 来源 App 拿到地图选择结果，继续配置回充或导览业务。
设置点位 用户在地图上新增、查看、编辑、删除点位，并保存点位的位置和朝向。 当前地图带有可被业务 App 使用的点位列表。
导航验证点位 用户选择一个点位，让机器人前往该点位，确认点位是否可达且朝向是否正确。 用户知道这个点位能不能走到，失败时知道原因。
维护地图 用户查看已有地图，必要时小范围修正底图或点位。 保存后的地图继续用于下载、选图和业务 App 调用。

3. 竞品调研

调研来源：建图工具竞品调研

4. 关键取舍

取舍项 本期口径
建图方式 手动建图。用户通过实体手柄为主、网页控制台为备用来移动机器人。
地图底图 只处理没扫到、可通行、障碍物三类底图结果。扫描点、轨迹、比例网格和机器人位置是辅助显示。
地图编辑 支持已保存地图的小范围底图修正。大面积没扫到、整体变形或位置明显错位时重新新建地图。
点位 点位保存坐标和朝向；不配置路线、讲解、回充动作或导航策略。
单点导航验证 导航到点位使用点位坐标和朝向，验证点位是否可达且朝向是否正确。遇到障碍物时停止本次验证，不自动绕行，不等待通行。

5. 关键流程

流程 主线 关键分支
建图并下载 地图列表 → 新建地图 → 选择栅格精度 → 建图前检查 → 建图工作台 → 保存地图 → 地图列表 → 下载地图包 1. 检查失败：提示失败项，可重试或返回地图列表。<br>2. 建图中取消：二次确认后丢弃未保存地图。<br>3. 保存失败：停留建图工作台，保留未保存地图，允许重试或取消。<br>4. 需要修图或补点位：保存时选择保存并编辑，保存编辑后再下载。
业务 App 选图 自动回充 App 或导览 App 打开建图 App → 地图列表 → 选择已有地图 → 返回来源 App 1. 无可用地图：先新建并保存，保存后回到地图列表，再由用户选择地图返回来源 App。<br>2. 地图需要点位：进入点位配置，保存后回到地图列表，再选择地图返回来源 App。<br>3. 无法读取地图：不可选择。<br>4. 用户取消：返回未选择结果。
设置并导航验证点位 地图查看或地图编辑 → 设置点位 → 当前位置设点或地图选点 → 设置朝向 → 编辑名称、编号 → 保存编辑 → 选择点位 → 导航到点位验证 1. 点位落在没扫到、障碍物或地图外，或未设置朝向：不允许保存，定位到问题点位。<br>2. 导航验证开始条件不满足：不开始验证，提示对应原因。<br>3. 导航验证遇到障碍物、手柄接管、定位丢失、急停或连接断开：停止本次验证并提示原因。
地图维护 地图列表 → 地图查看 → 编辑底图或设置点位 → 保存编辑 → 地图列表 1. 未保存离开：二次确认，确认后丢弃未保存修改。<br>2. 点位不合法：不能保存点位，保留未保存状态。<br>3. 保存失败：保留未保存修改，允许重试或取消。

第三部分：功能详情（How）

F01 地图列表

覆盖子功能：F01-01 展示地图列表、F01-02 展示地图信息、F01-03 点击地图进入查看、F01-04 地图快捷操作、F01-05 删除地图、F01-06 处理无法读取地图、F01-07 刷新地图列表。

用户任务说明：用户打开建图 App 后，在地图列表中查看当前机器人的本地地图，并从这里进入查看、编辑、设置点位、下载、删除或刷新。

产品规则：

  • 地图列表展示地图名称、备注、最近保存时间、文件状态和缩略图。
  • 正常地图：可查看、编辑、设置点位、下载、删除，也可被业务 App 选择。
  • 无法读取地图：只支持删除，且不作为业务 App 可选地图。
  • 删除地图前必须二次确认；确认文案提示删除后，已选择该地图的业务 App 可能无法继续使用该地图。

F02 新建地图

覆盖子功能：F02-01 打开新建地图、F02-02 填写地图名称和备注、F02-03 生成默认地图名称、F02-04 校验地图名称、F02-05 放弃新建确认、F02-06 选择栅格精度。

用户任务说明：用户新建地图时填写地图名称和备注，选择本次建图使用的栅格精度；确认后进入建图前检查。

产品规则：

  • 地图名称不能为空。
  • 地图名称不能和当前机器人已有地图重名。
  • 默认地图名称格式：建图_YYYYMMDD_HHmmss_NNNN。
  • 栅格精度可选 0.03m、0.05m、0.10m；默认 0.05m。
  • 开始建图后，当前地图的栅格精度不可修改。
  • 用户已填写内容后关闭新建地图，需要确认是否放弃。

F03 建图前检查

覆盖子功能：F03-01 执行建图前检查、F03-02 检查失败提示、F03-03 检查失败后重试或返回。

用户任务说明：用户进入建图工作台前，系统先检查机器人和建图能力是否可用；通过后进入工作台，失败时告诉用户原因和处理动作。

检查项：

检查项 失败时提示
机器人控制可用 机器人控制不可用，请检查连接或控制服务。
底盘 / 急停正常 底盘或急停状态异常，请恢复后重试。
扫描数据正常 暂未收到扫描数据，请检查雷达或传感器。
建图服务正常 建图服务异常，请重试或联系支持。
本地保存能力正常 本地保存不可用，请检查存储空间或权限。

F04 建图工作台

覆盖子功能：F04-01 显示机器人位置和朝向、F04-02 显示地图生成过程、F04-03 显示轨迹和扫描点、F04-04 地图缩放、平移、适配和比例网格、F04-05 手柄控制机器人移动、F04-06 网页控制台移动机器人、F04-07 转向控制规则、F04-08 控制权和安全停止、F04-09 距离障碍物过近自动停止、F04-10 取消本次建图。

用户任务说明：用户在建图工作台通过手柄或网页控制台移动机器人，同时观察机器人位置、轨迹和地图生成过程，完成后保存地图或取消本次建图。

地图显示：

  • 画布显示没扫到、可通行、障碍物三类底图。
  • 辅助显示：机器人位置和朝向、轨迹、扫描点、比例网格。
  • 用户可缩放、平移、适配视图，并开关比例网格。

移动控制：

场景 规则
主控制 实体手柄。
备用控制 网页控制台，支持按钮点击和键盘快捷键。
按钮平移 按住移动，松开停止。
按钮转向 单击转 45°；长按按 15°/s 连续转，松开停止。
键盘快捷键 W / ↑ 前进；S / ↓ 后退；A 左移；D 右移；Q / ← 左转；E / → 右转；Space / Esc 停止。
键盘生效条件 控制台展开、页面获得焦点，且用户未输入地图名称、备注、点位字段或弹窗内容。
键盘平移 按下移动，松开停止。
键盘转向 点按转 45°；长按按 15°/s 连续转，松开停止。
控制权 手柄优先；手柄有控制权时，网页控制台按钮和键盘快捷键都不生效。
安全停止 超时、失焦、切页、控制台收起、连接断开、按下停止键或距离障碍物过近时停止网页控制。

退出规则：

  • 未开始建图时退出，直接返回地图列表。
  • 已开始建图但未保存时退出，需要二次确认；确认后丢弃本次未保存地图。

F05 保存地图

覆盖子功能：F05-01 保存地图、F05-02 保存并编辑、F05-03 保存前校验、F05-04 生成可用地图、F05-05 保存失败重试。

用户任务说明：用户完成建图后保存地图；也可以选择保存并编辑，保存成功后直接进入地图编辑页处理局部底图或点位问题。

保存前校验：

  • 地图名称可用。
  • 本次建图已生成有效地图。
  • 本地保存能力可用。
  • 没有阻断保存的建图异常。

保存结果：

产物 用户用途
地图底图 用户可查看、编辑、下载或被业务 App 选择。
缩略图 用户在地图列表中识别地图。
文件状态 决定地图是否可查看、下载和被业务 App 选择。
点位列表 与地图绑定；没有点位时为空。
点位更新时间 业务 App 判断是否需要重新读取点位。

保存失败：

  • 保存失败时停留在建图工作台，不丢失当前未保存地图。
  • 用户可重试保存，也可取消本次建图。

F06 地图查看

覆盖子功能：F06-01 查看底图和点位、F06-02 查看机器人位置、F06-03 地图缩放、平移、适配和比例网格、F06-04 查看页入口操作、F06-05 查看页禁止移动机器人。

用户任务说明：用户打开已有地图后，在查看页确认底图、已保存点位和机器人当前位置，并从这里继续进入编辑、设置点位、下载或返回列表。

产品规则：

  • 画布显示底图、已保存点位标记、机器人位置和朝向。
  • 支持缩放、平移、适配视图和比例网格开关。
  • 查看页操作：编辑、设置点位、下载地图、返回地图列表。
  • 查看页不提供移动机器人控制。

F07 编辑地图底图

覆盖子功能：F07-01 进入底图编辑、F07-02 设置可通行、F07-03 设置障碍物、F07-04 设置没扫到、F07-05 画笔大小和连续编辑、F07-06 保存底图编辑、F07-07 未保存离开确认、F07-08 撤销和重做。

用户任务说明：用户在已保存地图上做小范围底图修正；小问题用编辑工具处理，大面积没扫到、整体变形或位置明显错位时重新新建地图。

编辑工具：

工具 用途
设置可通行 把误标为障碍物或没扫到的小区域改成可走。
设置障碍物 补全漏扫墙体或固定障碍物。
设置没扫到 把用户不能确认的区域标回没扫到。

产品规则：

  • 画笔大小：1 ~ 8 个栅格。
  • 支持点击和拖动连续编辑。
  • 支持撤销和重做。
  • 保存编辑后更新缩略图和最近保存时间。
  • 未保存离开时弹窗确认；确认离开后丢弃未保存修改。

F08 点位配置

覆盖子功能：F08-01 展示点位列表、F08-02 展示地图点位标记、F08-03 地图点位和列表联动、F08-04 使用机器人当前位置新增点位、F08-05 在地图上选点新增点位、F08-06 连续新增点位、F08-07 编辑点位名称、编号和朝向、F08-08 编辑点位位置、F08-09 点位页移动机器人控制台、F08-10 删除点位、F08-11 点位合法性校验、F08-12 保存点位修改。

用户任务说明：用户在当前地图上配置点位的位置和朝向，用于后续给自动回充 App 或导览 App 继续配置业务内容。点位导航验证见 F11。

F08-01 / F08-02 查看点位

看什么：

  • 地图上展示已保存点位标记。
  • 右侧列表展示当前地图的点位。
  • 没有点位时，列表展示空状态。

F08-03 点位联动

怎么选：

  • 点击地图点位，列表中对应点位卡片高亮。
  • 点击列表卡片，地图上对应点位高亮。
  • 点击已选中的点位卡片头部，收起卡片详情并取消选中。

F08-04 / F08-05 / F08-06 新增点位

新增方式：

方式 规则 适用场景
使用机器人当前位置设点 每点击一次生成一个点位，读取机器人当前地图坐标。 机器人已移动到目标位置。
在地图上选点 开启后地图光标显示选点状态，点击可通行区域生成点位；长按点位标记并拖拽出方向，松开后写入朝向。 用户能从地图判断目标位置。
  • 地图选点开启后，可连续点击可通行区域新增点位，直到用户关闭选点状态。
  • 点击没扫到、障碍物或地图外区域时，不创建点位，并提示只能放在可通行区域。
  • 使用机器人当前位置设点时，点位朝向默认使用机器人当前朝向。
  • 在地图上选点时，创建点位后需要长按点位标记并拖拽设置朝向。
  • 拖拽过程中显示方向线和朝向预览，松开后保存到该点位。
  • 已创建点位也可长按点位标记拖拽调整朝向。

F08-07 点位字段

填什么：

字段 规则
ID 系统生成。
编号 默认 P001、P002、P003 递增；可修改；同一地图内不可重复；不区分大小写。
名称 默认 点位 N；可修改；不能为空。
坐标 保存和返回都使用地图坐标。
朝向 保存和返回都使用地图朝向。当前位置设点默认使用机器人当前朝向；地图选点后通过长按拖拽设置。
来源 机器人当前位置或地图选点。

F08-08 编辑点位位置

怎么改：

  • 位置编辑支持三种方式：拖动地图标记、使用机器人当前位置更新、地图重选。
  • 选中点位后，在点位卡片内展开详情。
  • 编号和名称可在卡片内编辑；朝向通过地图上长按点位标记拖拽调整。
  • 位置编辑需要点击编辑后再展开，避免误操作。

F08-09 点位页移动机器人控制台

用途：

  • 点位页移动机器人控制台只用于调整机器人位姿；按钮点击和键盘快捷键规则复用 F04 移动控制。

F08-10 删除点位

删除规则：

  • 删除点位前弹窗确认；确认后从地图和列表同步移除，并清空该点位选中态。

F08-11 点位合法性校验

校验什么：

  • 点位必须有地图坐标。
  • 点位必须有朝向。
  • 点位必须落在可通行区域。
  • 点位不能落在没扫到、障碍物或地图外。
  • 保存点位前一次性校验全部点位。
  • 校验失败时，提示问题原因并定位到问题点位。

F08-12 保存点位修改

保存规则：

  • 地图编辑页只有一个保存入口：保存编辑。
  • 只有底图修改时，保存底图并更新缩略图和最近保存时间。
  • 只有点位修改时，先校验全部点位，再保存点位。
  • 底图和点位都有修改时，先保存底图，再校验并保存点位。
  • 若底图已保存但点位不合法，页面展示“底图已保存，点位未保存”，保留点位未保存状态，并定位到问题点位。

F09 下载地图包

覆盖子功能：F09-01 下载入口、F09-02 下载地图包、F09-03 下载失败提示。

用户任务说明：用户从地图列表、地图详情弹窗或地图查看页下载地图包，用于二开交付、调试或导入自有系统。

下载包组成：

内容 说明
地图文件 当前地图底图文件。
点位列表 点位 ID、编号、名称、地图坐标、朝向、所属地图 ID。没有点位时为空列表。
元数据 地图 ID、名称、分辨率、坐标信息、生成时间、点位更新时间。

下载失败：

  • 下载失败时停留当前页面，提示失败原因。
  • 用户可重新触发下载。

F10 业务 App 选图与返回

覆盖子功能：F10-01 来源 App 打开地图选择页、F10-02 选择已有地图返回、F10-03 无地图时引导新建、F10-04 调用中新建后回到地图选择页、F10-05 取消选择返回、F10-06 禁止选择无法读取地图、F10-07 返回点位数量和更新时间。

用户任务说明：用户从自动回充 App 或导览 App 进入建图 App 选择地图；没有可用地图时先新建，保存后回到地图选择页，再选择地图返回来源 App。

返回口径：

调用方 返回内容 调用方继续做什么
自动回充功能 选择结果、地图 ID、地图名称、文件状态、最近保存时间、点位数量、点位更新时间。 继续设置回充点、回充参数和回充测试。
导览 App 选择结果、地图 ID、地图名称、文件状态、最近保存时间、点位数量、点位更新时间。 继续配置导览点、讲解、路线和试跑。
二开下载 地图文件、点位列表、元数据。 在二开环境中解析或导入使用。

点位数据对外口径：

  • 点位列表若被业务 App 读取，位置和朝向以地图坐标系为准，包含 x_m、y_m、theta_deg 和所属地图 ID。
  • 页面中的百分比坐标仅用于用户理解点位在画布上的相对位置。
  • 来源 App 可继续基于点位配置回充、导览、路线或讲解业务内容。

F11 点位导航验证

覆盖子功能：F11-01 选择待验证点位、F11-02 发起导航到点位、F11-03 展示导航状态和停止、F11-04 处理导航异常、F11-05 展示验证结果。

用户任务说明：用户选中一个已保存点位后，让机器人前往该点位并对齐朝向，验证点位是否可用。该能力只做单点验证，不保存路线，不配置导航策略。

F11-01 选择待验证点位

开始条件：

  • 点位在可通行区域。
  • 点位有地图坐标。
  • 点位有朝向。
  • 机器人已定位在当前地图。
  • 导航服务可用。
  • 手柄未占用控制权。
  • 当前没有正在执行的网页控制或键盘控制。

F11-02 / F11-03 发起和停止导航

执行规则：

  • 点击“导航到点位”后，机器人按点位坐标和朝向前往目标位置。
  • 执行中展示“前往点位中”，用户可随时停止。
  • 执行中禁止编辑点位、网页控制台和键盘控制。

F11-04 / F11-05 异常和结果

结果处理：

  • 到达点位且朝向对齐：提示“已到达点位”。
  • 到达点位但朝向未对齐：提示“已到达附近，朝向未对齐”。
  • 用户主动停止：提示“已停止前往点位”。
  • 手柄接管：提示“手柄已接管，已停止前往点位”。
  • 障碍物阻挡：提示“前方有障碍物，已停止前往点位”。
  • 无法规划路径：提示“无法到达该点位，请调整点位或检查地图”。
  • 定位丢失、急停、连接断开或导航服务失败：提示对应原因，允许重试。
  • 验证失败不修改点位数据。
